Setting up the Project:
1.	Open your favorite IDE, preferably Eclipse.
2.	Go to File>import>maven>existing maven project, Select the project.
3.	Once the project is created, update the maven and built the project.


Execution:
1.	Run the project as Spring Boot App.
2.	Hit the URL form the browser -> http://localhost:8080/index (port is subjected to installed server)
3.	Insert the text over the textbox, and hit the submit button.
4.	Top 10 hashtags with decreasing order along with the number of times they appear in the text are printed.
